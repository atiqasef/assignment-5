Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent:

  • 1. innerHTML works with an HTML element like various tag. example: p tag, (h1-h6) tag, span tag etc
  • 2. innerText is work for change or set dynamic text for an HTML element in DOM
  • 3. textContent is similar as innerText . But we should textContent because of it's useability . It don't show any error like innerText.
